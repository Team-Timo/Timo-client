# Skill Index

## Git

| 스킬 | 트리거 | 설명 |
|------|--------|------|
| `timo-commit` | "커밋해줘" | diff를 기능 단위로 분해하고 커밋 메시지 작성 |
| `timo-pr` | "PR 작성해줘" | 커밋 히스토리 기반으로 PR 템플릿 채우기 |
| `timo-issue` | "이슈 만들어줘" | 기능/버그 이슈 템플릿 채우기 + 브랜치명 제안 |

## UI

| 스킬 | 트리거 | 설명 |
|------|--------|------|
| `timo-component` | "컴포넌트 만들어줘" | TypeScript + Tailwind 컴포넌트 생성 |
| `timo-page` | "페이지 만들어줘" | 라우트 설계부터 컴포넌트 조합까지 |
| `timo-storybook` | "스토리 만들어줘" | 컴포넌트 분석 후 Story 파일 생성 |

## Quality
